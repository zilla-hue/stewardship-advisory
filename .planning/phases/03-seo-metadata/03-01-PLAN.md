**1. Install react-helmet-async:**
```bash
npm install react-helmet-async
```

**2. Create `src/lib/seo.tsx` — reusable PageSEO component:**

Define a SITE_URL constant set to `"https://stewardshipadvisory.com"` (the canonical production URL — will be correct once deployed; OG tags require absolute URLs).

Define a SITE_NAME constant: `"Stewardship Advisory"`.

Define a DEFAULT_OG_IMAGE constant: `"/image/og-default.png"`.

Create and export a `PageSEO` component that accepts these props:
- `title: string` — page-specific title (will be composed as `{title} | Stewardship Advisory`)
- `description: string` — meta description for the page
- `path: string` — the route path (e.g., "/about") for constructing og:url
- `ogImage?: string` — optional override for og:image (defaults to DEFAULT_OG_IMAGE)
- `isHome?: boolean` — when true, uses the title as-is without appending site name

The component renders a `<Helmet>` with:
- `<title>{isHome ? title : title + " | " + SITE_NAME}</title>`
- `<meta name="description" content={description} />`
- `<meta property="og:title" content={computed title} />`
- `<meta property="og:description" content={description} />`
- `<meta property="og:image" content={SITE_URL + (ogImage || DEFAULT_OG_IMAGE)} />`
- `<meta property="og:url" content={SITE_URL + path} />`
- `<meta property="og:type" content="website" />`
- `<meta property="og:site_name" content={SITE_NAME} />`
- `<link rel="canonical" href={SITE_URL + path} />`

**3. Wrap HelmetProvider in `src/main.tsx`:**
Import `HelmetProvider` from `react-helmet-async`. Wrap it OUTSIDE `BrowserRouter`:
```tsx
<StrictMode>
  <HelmetProvider>
    <BrowserRouter>
      <App />
    </BrowserRouter>
  </HelmetProvider>
</StrictMode>
```

**4. Update `index.html` base meta tags:**
Add these fallback meta tags to the `<head>` section (Helmet will override per-page, but crawlers that don't execute JS see these):
- `<meta name="description" content="Stewardship Advisory is an institutional communications and reputation advisory practice supporting leaders and organisations in the public eye." />`
- `<meta property="og:title" content="Stewardship Advisory | Institutional Communications & Reputation Advisory" />`
- `<meta property="og:description" content="Independent strategic counsel for institutional leaders and organisations navigating public trust, stakeholder engagement, and reputation." />`
- `<meta property="og:image" content="https://stewardshipadvisory.com/image/og-default.png" />`
- `<meta property="og:type" content="website" />`
- `<meta property="og:site_name" content="Stewardship Advisory" />`

**5. Generate branded OG image (1200x630px):**
Create `scripts/generate-og-image.cjs` — a Node.js script that generates the OG image using the `canvas` npm package (install as devDependency: `npm install -D canvas`).

The script should:
- Create a 1200x630 canvas
- Fill background with #0A1628 (the brand navy)
- Add a subtle decorative element: a large circle with very low opacity (#FFFFFF at 3-5% opacity) in the upper-right area, similar to the ambient glow in the Hero section
- Draw the firm name "STEWARDSHIP" in white, bold, centered, large font (~60px, using system sans-serif since canvas doesn't have Inter — "Arial" or "Helvetica" as fallback)
- Draw "ADVISORY" below it in slightly smaller or same size, with some letter-spacing feel
- Draw a thin horizontal line (~2px, white at 20% opacity) separating the name from the tagline
- Draw tagline below the line: "Institutional Communications & Reputation Advisory" in white at 60% opacity, ~22px
- Output to `public/image/og-default.png`

Run the script after creating it:
```bash
node scripts/generate-og-image.cjs
```

Verify the generated image exists and is approximately 1200x630:
```bash
file public/image/og-default.png
```

If `canvas` npm package fails to install (it requires native dependencies), fall back to creating the OG image using a simpler approach — generate an SVG string and convert it to PNG using `sharp` (install as devDependency: `npm install -D sharp`). The SVG approach:
- Create an SVG string with the same design (navy background, white text, tagline)
- Use sharp to convert the SVG buffer to a 1200x630 PNG
- Output to `public/image/og-default.png`
